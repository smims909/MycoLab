# 🍄 MycoLab (v4.8)
**Système de Gestion et d'Analyse pour l'Expérimentation Mycologique**

MycoLab est un outil de laboratoire minimaliste conçu pour les chercheurs et expérimentateurs qui exigent une traçabilité rigoureuse. L'application est optimisée pour transformer des protocoles théoriques en données exploitables.

🔗 **Accès Direct :** [https://smims909.github.io/MycoLab/](https://smims909.github.io/MycoLab/)

---

## ⚡ Installation & Utilisation
L'application est "Zero-Install" et fonctionne sans serveur.
1. **Téléchargement :** Enregistrez le fichier `index.html` sur votre appareil.
2. **Exécution :** Ouvrez simplement `index.html` avec votre navigateur (Chrome, Firefox, Safari).

*Note : Une fois le fichier enregistré localement, l'outil fonctionne 100% hors-ligne.*

---

## 🔬 Approche Expérimentale
* **Rigueur de Mesure :** Conversion dynamique entre **Volume (L)** et **Masse (Kg)** basée sur la masse volumique des intrants pour garantir la reproductibilité des substrats.
* **Suivi Thermique :** Enregistrement des températures réelles pour corréler le développement biologique et les conditions environnementales.
* **Indice de Réussite (IR) :** Analyse quantitative de la performance de chaque lot pour isoler les meilleurs protocoles.
* **Confidentialité Totale :** Aucune donnée ne quitte le laboratoire. Tout est stocké localement dans votre navigateur (LocalStorage).

## 🛠 Fonctionnalités Clés
- [x] **Éditeur de Protocoles :** Organisation structurée par phases de culture.
- [x] **Journal de Bord :** Chronologie d'événements avec horodatage automatique.
- [x] **Snapshot System :** Sauvegarde miroir automatique pour prévenir la perte de données expérimentales en cas de vidage du cache.
- [x] **Export PDF :** Génération de fiches de synthèse compactes optimisées pour l'archivage physique au laboratoire.
