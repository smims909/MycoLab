# MycoLab

Calculateur de substrats mycologiques. Gestion dynamique des volumes et des ratios.

## Accès

https://smims909.github.io/MycoLab/

Application progressive (PWA) : installation possible sur l'écran d'accueil pour un usage hors ligne.

## Principes

- Interface directe pour une manipulation rapide.
- Calcul automatique des quantités en fonction du volume cible.
- Données stockées localement (aucun serveur).

## Fonctionnalités

- Ajustement en temps réel des proportions.
- Équilibrage automatique des ratios à 100 %.
- Export PDF des fiches de calcul.
- Mode sombre intégré.
- Sauvegarde et restauration des recettes (format JSON).
- Interface bilingue français/anglais.
