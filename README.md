# 🍄 MycoLab (v4.8)
**Système de Gestion et d'Analyse pour l'Expérimentation Mycologique**

MycoLab est un outil de laboratoire minimaliste conçu pour les chercheurs et expérimentateurs qui exigent une traçabilité rigoureuse. L'application transforme vos protocoles théoriques en données exploitables.

---

## ⚡ Installation & Usage Rapide

L'application est "Zero-Install" et fonctionne directement dans votre navigateur.

1. **Accès :** Cliquez sur le [Lien Direct](https://smims909.github.io/MycoLab/).
2. **Usage Hors-ligne :** Pour l'utiliser au laboratoire sans connexion, enregistrez simplement la page (`Fichier > Enregistrer sous`) sur votre appareil. Le fichier `index.html` fonctionnera sans Wi-Fi.
3. **Version Mobile :** Ouvrez le lien sur votre téléphone et choisissez **"Ajouter à l'écran d'accueil"** dans les options du navigateur pour l'utiliser comme une application native.

---

## 🔬 Approche Expérimentale
* **Rigueur de Mesure :** Conversion dynamique **Volume (L)** / **Masse (Kg)** basée sur la densité des intrants pour une reproductibilité parfaite.
* **Suivi Thermique :** Enregistrement des températures réelles pour corréler le développement biologique et l'environnement.
* **Indice de Réussite (IR) :** Analyse quantitative de la performance de chaque lot pour isoler les meilleurs protocoles.
* **Confidentialité Totale :** 100% local. Aucune donnée ne quitte le laboratoire. Tout est stocké dans votre navigateur (LocalStorage).

## 🛠 Fonctionnalités Clés
- [x] **Éditeur de Protocoles :** Organisation structurée par phases de culture.
- [x] **Journal de Bord :** Chronologie d'événements avec horodatage automatique.
- [x] **Snapshot System :** Sauvegarde miroir automatique pour prévenir la perte de données.
- [x] **Export PDF :** Fiches de synthèse compactes optimisées pour l'archivage physique.
