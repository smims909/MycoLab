# 🍄 MycoLab (v5.3)
**Système de Gestion et d'Analyse pour l'Expérimentation Mycologique**

MycoLab est un outil de laboratoire minimaliste conçu pour les chercheurs et expérimentateurs qui exigent une traçabilité rigoureuse. L'application transforme vos protocoles théoriques en données exploitables via une interface Gantt dynamique.

---

## ⚡ Accès Direct
L'application est sans installation et fonctionne directement dans votre navigateur.

> **Lien officiel :** https://smims909.github.io/MycoLab/

---

## 🔬 Approche Expérimentale
* **Chronologie Gantt Dynamique :** Visualisation continue des lots sur toute l'année avec indicateurs d'état codifiés (Incubation, Observation, Colonisation, Récolte).
* **Rigueur de Mesure :** Conversion dynamique **Volume (L)** / **Masse (Kg)** basée sur la densité des intrants pour une reproductibilité parfaite.
* **Suivi Thermique :** Enregistrement des températures réelles pour corréler le développement biologique et l'environnement.
* **Indice de Réussite (IR) :** Analyse quantitative de la performance de chaque lot pour isoler les meilleurs protocoles.
* **Confidentialité Totale :** 100% local. Aucune donnée ne quitte le laboratoire. Tout est stocké dans votre navigateur (LocalStorage).

## 🛠 Usage & Installation
- **Usage Hors-ligne :** Pour l'utiliser au laboratoire sans connexion, enregistrez simplement la page (`Fichier > Enregistrer sous`). Le fichier `index.html` fonctionnera sans Wi-Fi.
- **Version Mobile :** Ouvrez le lien sur votre téléphone et choisissez **"Ajouter à l'écran d'accueil"** pour l'utiliser comme une application native (PWA).
- **Export PDF :** Génération de fiches de synthèse compactes optimisées pour l'archivage physique.

## 💾 Sauvegarde & Sécurité
Toutes les données sont stockées dans le cache de votre navigateur. Pour une sécurité maximale :
1. Utilisez l'onglet **Sauvegarde** régulièrement.
2. Copiez vos données ou téléchargez le fichier JSON de backup.
3. En cas de changement de navigateur ou de nettoyage de cache, vous pourrez réimporter votre historique instantanément.
