# 🍄 MycoLab (v4.8)
**Système de Gestion et d'Analyse pour l'Expérimentation Mycologique**

MycoLab est un outil de laboratoire minimaliste conçu pour les chercheurs et expérimentateurs qui exigent une traçabilité rigoureuse. L'application transforme vos protocoles théoriques en données exploitables.

🔗 **Accès Direct :** [https://smims909.github.io/MycoLab/](https://smims909.github.io/MycoLab/)

---

## ⚡ Guide d'Installation Rapide

L'application est "Zero-Install". Voici comment l'utiliser en moins de 60 secondes :

### 1. Utilisation Immédiate (En ligne)
Cliquez simplement sur le lien suivant : [Accès MycoLab](https://smims909.github.io/MycoLab/). L'interface s'ouvrira directement dans votre navigateur.

### 2. Utilisation Locale & Hors-ligne (Recommandé)
Pour utiliser MycoLab au laboratoire sans connexion internet :
* **Sur PC/Mac :** Faites un clic droit sur la page du lien ci-dessus et choisissez **"Enregistrer sous..."**. Enregistrez le fichier `index.html` sur votre bureau.
* **Exécution :** Double-cliquez sur le fichier enregistré. Il s'ouvrira dans votre navigateur (Chrome, Firefox, Safari) et fonctionnera même sans Wi-Fi.

### 3. Installation sur Mobile (iOS/Android)
* Ouvrez le lien dans votre navigateur mobile.
* Utilisez l'option **"Ajouter à l'écran d'accueil"** dans les réglages de votre navigateur. 
* L'application apparaîtra comme une application native sur votre téléphone.

## 🔬 Approche Expérimentale
* **Rigueur de Mesure :** Conversion dynamique entre **Volume (L)** et **Masse (Kg)** basée sur la masse volumique des intrants pour garantir la reproductibilité.
* **Suivi Thermique :** Enregistrement des températures réelles pour corréler le développement biologique et l'environnement.
* **Indice de Réussite (IR) :** Analyse quantitative de la performance de chaque lot pour isoler les meilleurs protocoles.
* **Confidentialité Totale :** Aucune donnée ne quitte le laboratoire. Tout est stocké localement dans votre navigateur (LocalStorage).
